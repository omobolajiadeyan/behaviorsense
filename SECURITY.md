# Security Policy

BehaviorSense is a portfolio-grade security tool and may produce false positives
or false negatives. Treat its output as triage evidence, not as an automated
incident declaration.

## Reporting Vulnerabilities

Please report security issues privately through GitHub security advisories or by
contacting Omobolaji Adeyan through the profile links at:

https://github.com/omobolajiadeyan

Do not open public issues for vulnerabilities that could expose users or test
environments.

## Data Handling

BehaviorSense runs locally and does not send log data to external services.
Avoid committing real production logs, credentials, IP addresses, usernames, or
customer data to issues, pull requests, examples, or tests.

## Supported Versions

The `main` branch is the only supported development line until tagged releases
are published.
